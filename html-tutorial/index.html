HTML Tutorial

0. Introduction

This is a simple tutorial that will get you started quickly with HTML.

Although simple, it's meant to be quite comprehensive. You will do things quite close to what professionals do. Anyway, we encourage you to widen your knowledge by reading the resources we include at the end.

1. Preparation

1.1 Download an editor

In order to follow this tutorial you need a code text editor. I recommend Brackets but you could use others like Sublime Text, Notepad++, Atom, etc.

1.2 Download the startup files

After installing your preferred editor, create a folder in your computer. That's where you will add your project files. For now, add these two files inside that folder (right click and save): index.html and main.css.

1.3 Open the files in the editor and browser

Open the folder in your editor. You can drag and drop the folder from the file explorer to the editor. Then open the index.html file. For now we will just edit this file.

Now open the index.html in a browser. If you use Brackets you can use the Live Preview feature: after opening index.html select File > Live Preview (or click the "lightning bolt" icon)

So now you should see the HTML code in the editor and the page in the browser, that should look like this:

It should look pretty similar. If you don't see the title in green color maybe forgot to download the main.css file.

2. HTML elements and tags

In the code of index.html you should now focus on the bottom part, that looks like this: 

    
      
      
My HTML page

      
I will use this page to practice HTML.

Those are HTML elements. Notice that they are placed between <body> and </body>. What you put inside there is displayed in the browser.

The line 17 is a comment. That's ignored by the browser, so it's not displayed actually.

The line 18 is a header. Notice the <h1> and </h1> that surround the title text.

The line 19 is a paragraph. Notice the <p> and </p>.

As you see, HTML elements are surrounded by those strange pieces of text like <h1> or <p>. Those are called tags. Each tag has a meaning. There are a number of them; for now, let's learn:

  • h1 is a big header
  • h2 is slightly smaller header
  • h3 is an enven smaller header
  • p is a paragraph
  • img is an image

If you use Brackets Live Preview you will already see the changes in your browser. Otherwise, save the index.html file and reload the page in your browser.

You should see the smaller header below the paragraph, and below that, an image.

Notice that the img element only uses one tag. Most elements use an opening and a closing tag to surround the content of the element, but a few elements like img use only the opening tag.

Also, notice that the img element indicates the address of the image using src="image url address" inside the tag. That's an attribute of the img tag.

Each tag supports a number of attributes. If you use any, you need to write them inside the opening tag, separated by spaces. Some attributes are required and others optional. The src attribute is required for the img element. On the other side, the width attribute is optional. If you remove it, the image will still be displayed, but with the original size.
