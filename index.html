Code then...
We teach you what you need to know about code, then what comes next is up to you. The possibilities are endless.

Philosophy_

  • Coding is fun, useful, creative, rewarding and anyone can learn it given enough interest.
  • Go into detail about programming to achieve a strong knowledge of core concepts.
  • Stress the importance of good code: indentation, meaningful names, documentation, avoid repetition, tests, ...
  • Two classes per week so you can better assimilate what you learn.
  • Exercises to practice at home, with support from us when necessary.
  • Up-to-date technologies that are in demand: git, flexbox, sass, angular, Java 8, reactive streams, SQL, No SQL, ...
  • Comparison between different languages (Java, JavaScript, Ruby, Python, Scala) to better understand the core concepts of coding.
  • We care about what you learn.

About how the contents are organized

Many courses start with frontend topics like HTML/CSS. That may be more visually appealing and (deceptively) easier to grasp, but we think that it is important to first build a solid foundation by learning the core programming concepts. With that knowledge in your toolbelt, you will be better prepared to learn any languages, tools or frameworks you need to reach your goals. Moveover, you will enjoy programming much more.
